# Responsive Portfolio Website

This is my responsive portfolio website created using pure HTML, CSS, and JavaScript.

## Features

- Responsive design for desktop, tablet, and mobile
- Hero, About, Skills, Projects, and Contact sections
- Google Font
- Smooth scrolling
- CSS Flexbox layout
- Hover effects
- JavaScript contact form validation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
index.html
styles.css
image.jpg
README.md
```
